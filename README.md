# FTP-using-Java


Creating File Transfer Protocol using Java Socket Programming

# This project is about implementing File Transfer Protocol using Java Socket Programming
## Server will run on port no 2001 and Client implemented using Java Swing

<br> <br>
## **NOTE**
- As of right now this will only work in Windows OS. As File path in Windows is different than Linux/ Unix and MacOS.
- Feel free to change the file path according to your Operating System.

### List of funtions
* List
* Download
* Upload
* Set the file as read only
* Delete a file
* Rename a file
* Create a directory
* File or Directory
* Get last updated information
* Set last updated information
